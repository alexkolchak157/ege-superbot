# Шаблон экспертных оценок для few-shot обучения

Заполните 3-5 примеров для каждого задания. Каждый пример — это реальный ответ ученика + оценка настоящего эксперта ЕГЭ.

**Как заполнять:**
- Берите реальные ответы учеников (или составляйте типичные)
- Включайте разные уровни: идеальный (3/3), средний (1-2/3), нулевой (0/3)
- Комментарии эксперта должны быть конкретными — именно они учат модель
