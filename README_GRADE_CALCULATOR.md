# Калькулятор градации оценок

Простое приложение для расчёта оценок в тесте при заданном количестве вопросов.

## Возможности

- **Российская 5-балльная система оценок** (5, 4, 3, 2)
- **Два режима оценивания:**
  - **Обычный режим (сдаёт ЕГЭ):** 5 (85%+), 4 (65-84%), 3 (50-64%), 2 (<50%)
  - **Режим "не сдаёт ЕГЭ":** 5 (75%+), 4 (55-74%), 3 (40-54%), 2 (<40%)
- **Подробная таблица** с количеством правильных/неправильных ответов для каждой оценки
- **Цветовая индикация** оценок для удобства

## Варианты запуска

### Вариант 1: GUI приложение (tkinter)

**Требования:**
- Python 3.6+
- tkinter (обычно идёт с Python)

**Установка tkinter (если не установлен):**

На Ubuntu/Debian:
```bash
sudo apt-get install python3-tk
```

На macOS (обычно уже установлен):
```bash
# tkinter идёт с Python из официального установщика
```

На Windows (обычно уже установлен):
```bash
# tkinter идёт с Python из официального установщика
```

**Запуск:**
```bash
python3 grade_calculator.py
```

### Вариант 2: Веб-приложение (HTML)

Просто откройте файл `grade_calculator.html` в любом браузере. Не требует установки Python или других зависимостей.

## Использование

1. Введите количество вопросов в тесте
2. Выберите режим оценивания (установите чекбокс "Не сдаёт ЕГЭ" для более мягких критериев)
3. Нажмите "Рассчитать"
4. Просмотрите таблицу с результатами

## Примеры

**Тест на 20 вопросов (обычный режим):**
- 17-20 правильных ответов (85-100%) → оценка 5
- 13-16 правильных ответов (65-80%) → оценка 4
- 10-12 правильных ответов (50-60%) → оценка 3
- 0-9 правильных ответов (<50%) → оценка 2

**Тест на 20 вопросов (режим "не сдаёт ЕГЭ"):**
- 15-20 правильных ответов (75-100%) → оценка 5
- 11-14 правильных ответов (55-70%) → оценка 4
- 8-10 правильных ответов (40-50%) → оценка 3
- 0-7 правильных ответов (<40%) → оценка 2
