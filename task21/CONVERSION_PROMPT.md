### ПРАВИЛА КОНВЕРТАЦИИ:

1. **ID задания**: Формат `task21_XXX`, где XXX — номер с ведущими нулями (001, 002, ...)

2. **image_url**: Формат `data/images/task21_КЛЮЧЕВОЕ_СЛОВО_XXX.png`
   - Используй транслитерацию названия рынка для имени файла
   - Примеры: `task21_designers_001.png`, `task21_smartphones_002.png`

3. **curve_shifted и shift_direction**:
   - Определи из текста, какая кривая сдвинулась (supply/demand)
   - Определи направление сдвига (right/left)
   - Сдвиг вправо = увеличение, влево = уменьшение

4. **question_1 (изменение цены)**:
   - Если предложение сдвигается ВПРАВО → цена УМЕНЬШИЛАСЬ
   - Если предложение сдвигается ВЛЕВО → цена УВЕЛИЧИЛАСЬ
   - Если спрос сдвигается ВПРАВО → цена УВЕЛИЧИЛАСЬ
   - Если спрос сдвигается ВЛЕВО → цена УМЕНЬШИЛАСЬ

5. **question_2 (фактор + объяснение)**:
   - Внимательно определи, про что вопрос (спрос или предложение)
   - change_direction должно соответствовать направлению сдвига на графике
   - example_answers — возьми из эталонных ответов или придумай релевантные

6. **question_3 (прогноз)**:
   - Внимательно прочитай условие
   - Определи ОБЕ переменные для прогноза (обычно спрос/предложение + цена)
   - Для каждой переменной определи правильный ответ


### ПРИМЕР ВХОДНЫХ ДАННЫХ:

```
Задание 21
На графике изображено изменение ситуации на рынке услуг дизайнеров интерьера.
Кривая предложения переместилась из положения S в положение S₁ при неизменном спросе D.
(На графике P – цена услуг; Q – количество услуг.)

Вопросы:
1. Как изменилась равновесная цена?
2. Что могло вызвать изменение предложения? Укажите любое одно обстоятельство (фактор) и объясните его влияние на предложение.
3. Как изменятся спрос и равновесная цена на данном рынке, если вырастет число граждан, купивших новые квартиры?

Эталонные ответы:
1. уменьшилась
2. количество производителей услуг: выросло количество людей, получивших специальность дизайнера в профильных вузах, следовательно, предложение услуг дизайнеров увеличилось
3. спрос увеличится, равновесная цена увеличится
```
